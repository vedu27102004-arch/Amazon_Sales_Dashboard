# 📊 Amazon Sales Dashboard (Excel Project)

## 📌 Overview

This project showcases an **Amazon Sales Dashboard created entirely in Microsoft Excel**, focused on transforming raw e-commerce data into meaningful insights through structured analysis and clear visualization. The goal of this project is to demonstrate how Excel can be used as a powerful data analysis and reporting tool without relying on advanced features like Pivot Tables or Slicers.

Instead, this dashboard emphasizes the use of **formulas, logical functions, and standard charting techniques** to build an interactive and informative reporting solution. It highlights key sales metrics, identifies trends, and provides a clear understanding of business performance.

---

## 🎯 Objectives

The main objectives of this project are:

* To analyze e-commerce sales data in a structured way
* To identify key trends in sales, categories, and regions
* To build a visually appealing and easy-to-understand dashboard
* To demonstrate Excel-based data analysis without Pivot Tables or Slicers
* To improve decision-making through clear data insights

---

## 📂 Dataset Description

The dataset used in this project contains typical e-commerce transaction details, including:

* Order ID
* Order Date
* Customer Information
* Product Category
* Product Name
* Quantity Sold
* Sales Amount
* City / Region
* Order Status

This data was first cleaned and organized to ensure consistency and accuracy before building the dashboard.

---

## 🧹 Data Cleaning & Preparation

Before creating the dashboard, the dataset underwent several preprocessing steps:

* **Handling Missing Values**
  Missing or null values were identified and handled appropriately.

* **Removing Duplicates**
  Duplicate records were checked and removed to maintain data integrity.

* **Formatting Columns**
  Dates, numbers, and text fields were standardized for consistency.

* **Creating Helper Columns**
  Additional columns such as:

  * Month
  * Year
  * Category-wise grouping
    were created using Excel formulas.

---

## 🧠 Data Analysis Approach

This project focuses on using **Excel formulas and functions** instead of Pivot Tables. Some commonly used techniques include:

* `SUM()` → Total sales calculation
* `AVERAGE()` → Average values
* `COUNT()` / `COUNTIF()` → Counting records
* `SUMIF()` / `SUMIFS()` → Conditional aggregation
* `IF()` → Logical conditions
* `INDEX()` + `MATCH()` → Dynamic lookups
* `MAX()` / `MIN()` → Identifying top and bottom values

These functions helped in extracting meaningful insights from the dataset.
